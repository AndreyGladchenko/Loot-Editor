Указать путь к файлам Loot

Вы можете переопределить папку Loot (содержащую папки CooldownGroups, Items, Nodes, Spawners). Все подпапки являются необязательными. Редактор будет читать и писать Override конфигурации из указанной папки, а Default системные файлы по-прежнему будут считываться из папки по умолчанию.

Пустой путь сбрасывает настройки на папку по умолчанию.
